# 終わった作業

このフォルダは、完了した作業を整理して保管するための場所です。

まずは土台だけを作ってあります。
難しい設定は入れていないので、必要になったら少しずつ増やせます。

## フォルダ構成

- `works/`
  完了した作業物を入れます。
- `notes/`
  作業メモ、引き継ぎメモ、簡単な説明を書きます。
- `tmp/`
  一時ファイルを置く場所です。Git には保存しません。

## 使い方

1. 完了した作業ごとに `works/` の中へフォルダを作る
2. 必要なら `notes/` に説明メモを残す
3. 一時ファイルは `tmp/` に置く
4. 区切りのよいタイミングで Git に記録する

例:

```text
works/
└── 2026-04_sample-task/
    ├── result.txt
    └── screenshot.png
```

## Git の基本操作

変更を確認:

```bash
git status
```

はじめて記録するとき:

```bash
git add .
git commit -m "初期設定"
```

## メモ

- `tmp/` の中身は Git に入らない設定です
- 空フォルダを残すために `.gitkeep` を使っています
